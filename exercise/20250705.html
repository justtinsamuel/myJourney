Introduction to Git

Pada terminal git apa saja yang umum digunakan:

  • git add . => untuk add semua file untuk git ke repo, titik bisa di ganti spesifik file yg mau di push
  • git commit -m "comment" => untuk leave notes commit
  • git push origin main => untuk push ke repo di git
  • git status => untuk cek apa saja yang sudah berubah di local file
